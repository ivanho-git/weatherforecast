## 🛠 Features

- 🔍 City-based weather search
- 🌈 Colorful, emoji-enhanced UI
- 🌡️ Temperature in Celsius
- 📝 Condition description
- 🖼️ Dynamic weather icon
- ✅ Status bar for feedback messages
- 📦 Lightweight & beginner-friendly
